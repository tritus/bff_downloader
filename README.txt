Pour faire marcher ça, il faut aller dans les extensions de chrome et
cliquer sur "charger l'extension non empaquetée". Si cette option est pas dispo,
c'est surement que le mode developpeur est pas activé. Je te laisse trouver 
comment l'activer le cas échéant. Il suffit ensuite de choisir le dossier de
l'extension.
Après, dès que tu navigues sur les pages de bff, le script scrute les pages.
Il suffit alors de raffraichir la page où tu veux faire apparaitre le lien
(je ne m'explique pas vraiment pourquoi il apparait pas direct... c'est surement à
cause du fonctionnement de bff). Après rafraichissement, là où il y avait
la date apparait le lien de download.
