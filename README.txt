Pour faire marcher ça, il faut aller dans les extensions de chrome et
cliquer sur "charger l'extension non empaquetée". Si cette option est pas dispo,

l'extension.
Après, dès que tu navigues sur les pages de bff, le script scrute les pages et
là où il y avait la date du post apparait le lien de download.
